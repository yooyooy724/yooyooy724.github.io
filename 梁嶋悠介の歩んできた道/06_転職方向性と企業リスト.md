# 転職方向性と企業リスト

これまでの経歴、スキル（Unity、C#、UI設計、MVVMアーキテクチャ等）、そして「遊び手ベース」という価値観やToCビジネスへの志向を踏まえ、考えられる転職の方向性と、それぞれの方向性に合致する具体的な企業リストを整理しました。

## 案A：ゲーム会社のUIエンジニア / UIデザイナー
Unityを用いたUI開発やグラフィック制作の実績を最大限に活かし、中規模〜大規模のゲームスタジオでUI専門職として活躍するルートです。ゲーム開発におけるUIの重要性は年々高まっており、UnityのUI実装に強いエンジニア・デザイナーは引く手あまたの状況です。

| 企業名 | 特徴・想定される役割 | 備考 |
| :--- | :--- | :--- |
| **株式会社Cygames** | 圧倒的なクオリティを誇るモバイルゲーム開発。UIデザイナー/エンジニアとして高いレベルでの制作が求められる環境。 | 大規模開発でのUI専門職 |
| **グリー株式会社** | モバイルゲームを中心に多様なタイトルを展開。エンジニアリングとデザインの橋渡し役としてUIエンジニアの需要が高い。 | 大規模開発・ToC |
| **株式会社コロプラ** | アクション性の高いモバイルゲームに強み。UIのインタラククション設計やUnityでの実装スキルが活かせる。 | モバイルゲーム・Unity |
| **株式会社DeNA** | ゲーム運営に強み（DeNA Games Tokyo等）。継続的なUI改善やユーザー体験向上のための実装が求められる。 | 運営・データドリブン |
| **株式会社アカツキ** | IPタイトルや自社オリジナルタイトルの開発。ユーザー感情を動かすUI/UX設計に重きを置いている。 | モバイルゲーム |

## 案B：インディーゲームスタジオ / スタートアップのゲーム開発職
企画から開発、UI、広告運用まで一人でこなした「フルスタックなゲーム開発者」としての経験が活きるルートです。裁量が大きく、ゲーム全体の面白さに直接関与できる環境が魅力です。

| 企業名 | 特徴・想定される役割 | 備考 |
| :--- | :--- | :--- |
| **株式会社MyDearest** | VRゲームスタートアップ。新しいプラットフォームでのUI/UX設計やUnity開発の知見が活かせる。 | VR・スタートアップ |
| **Thirdverse** | VR/ブロックチェーンゲーム開発。グローバル向けのタイトル開発において、幅広い開発経験が求められる。 | Web3・VR |
| **株式会社ポケットペア** | インディー精神を持ったゲーム開発スタジオ（代表作：パルワールド等）。少人数で裁量を持って開発できる環境。 | インディースピリット |
| **株式会社ゲームフリーク** | ポケモンシリーズの開発。大規模ながらも企画から実装まで幅広く関われるポジションがある。 | コンシューマー |
| **その他インディースタジオ** | 少数精鋭のチームで、UIからドメインロジックまで横断的に開発を担うポジション。 | フルスタック・高裁量 |

## 案C：ゲーム会社のゲームデザイナー / プランナー
『Idle Minertia』で培った「広告逆算設計」「遊び手ベースのゲームデザイン」の経験を活かすルートです。エンジニアリング（Unity、C#）の素養を持つプランナーは、仕様を実装レベルで理解できるため、チーム開発において非常に重宝されます。

| 企業名 | 特徴・想定される役割 | 備考 |
| :--- | :--- | :--- |
| **株式会社バンダイナムコスタジオ** | 大規模プロジェクトにおけるゲームデザイナー。エンジニア視点を持った企画立案が強みになる。 | 大規模・コンシューマー |
| **KLab株式会社** | モバイルオンラインゲームの企画・運営。KPI分析やイベント設計など、データに基づくプランニングが求められる。 | モバイル・運営 |
| **ガンホー・オンライン・エンターテイメント** | パズル＆ドラゴンズ等の運営。ユーザー体験を重視したゲームデザインやバランス調整に関わる。 | モバイル・長期運営 |
| **株式会社f4samurai** | RPG等のモバイルゲーム開発。世界観やストーリーと連動したシステム設計・プランニング。 | モバイル・RPG |
| **株式会社アソビズム** | 独創的なゲームデザイン（城とドラゴン等）。「遊び」の根本から企画・設計するポジション。 | 独自性・企画力 |

## 案D：テック系スタートアップのフロントエンド / UXエンジニア
ゲーム業界で培ったUI/UXの知見やインタラクション設計の技術を、Webサービスやモバイルアプリ（非ゲーム）に転用するルートです。ToCプロダクトのスタートアップでは、ユーザー体験を劇的に向上させるUIエンジニアの価値が高まっています。

| 企業名 | 特徴・想定される役割 | 備考 |
| :--- | :--- | :--- |
| **ToC向けアプリ開発スタートアップ** | マッチング、エンタメ、ライフスタイル系アプリなど。UnityでのUI設計思想をReact Native等に転用。 | ToC・UX重視 |
| **EdTech / ヘルスケア系ベンチャー** | ゲーミフィケーションを取り入れたサービス開発。ユーザーのモチベーションを維持するUI/UX設計。 | ゲーミフィケーション |
| **メタバース関連スタートアップ** | cluster（クラスター株式会社）など。Unityを用いた仮想空間プラットフォームのUI/フロントエンド開発。 | メタバース・Unity |
| **デザインファーム系企業** | Goodpatchなど。クライアントのプロダクト開発において、高度なUI実装とプロトタイピングを担う。 | UI/UX専門 |

## 今後のステップ
これらのリストをもとに、どの方向性（あるいは企業）に最も興味があるか、またご自身のポートフォリオ（特に『Idle Minertia』の実績）をどのように見せていくかを検討していくと良いでしょう。
