# yooyooy724.github.io

A modern, responsive one-page portfolio site.

## Run locally

Because this is a static site, you can open `index.html` directly, or use a local server:

```bash
python3 -m http.server 8080
```

Then open:

http://localhost:8080

## Files

- `index.html`: page structure and sections
- `styles.css`: visual design and responsive layout
- `script.js`: interactions, scroll spy, and debug overlay
